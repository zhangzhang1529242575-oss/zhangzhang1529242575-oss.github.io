## 核心原理

重复控制基于**内模原理（Internal Model Principle）**，专门用于跟踪/抑制**周期性信号**。其核心思想是：

> 如果一个周期为 $T$ 的周期性扰动出现了一次，它将在下一个周期再次出现——控制器"记住"上一个周期的误差，并在当前周期产生补偿。

在频域中，重复控制器的核心是一个**周期信号发生器**：

$$G_{rc}(s) = \frac{e^{-sT}}{1 - e^{-sT}}$$

这个传递函数在基频 $\omega_0 = \frac{2\pi}{T}$ 及其所有整数倍频率 $k\omega_0$ 处具有**无穷大增益**，从而实现对周期性信号的无静差跟踪或完全抑制。

## 离散实现

### 传统形式

在实际数字控制（DSP/微控制器）中，重复控制器的传统离散形式为：

$$G_{rc}(z) = \frac{z^{-N}}{1 - Q z^{-N}} \cdot C(z)$$

其中：
- $N = \dfrac{T}{T_s}$ —— 一个信号周期内的采样点数
- $Q$ —— 内模系数（略小于 1 的常数，如 0.95；或低通滤波器），用于增强鲁棒性（以微小稳态误差换取稳定裕度）
- $C(z)$ —— 补偿器（相位超前 + 幅值补偿）

### 问题：非整数 $N$

传统 RC 隐含假设 $N$ 为整数，即采样频率 $f_s$ 恰好是基频的整数倍。然而在实际变频驱动中，这一条件**几乎不可能满足**。当 $N$ 非整数时，$z^{-N}$ 无法直接实现，传统做法取整 $N \approx \text{round}(f_s/f_0)$，导致 RC 谐振峰偏离实际谐波频率，抑制性能大幅下降。

### 改进型：分数阶重复控制（朱元等，2024）

将 $N$ 分解为整数部分和分数部分：

$$N = N_0 + N_1, \quad N_0 = \lfloor N \rfloor, \quad N_1 \in [0, 1)$$

采用 **Lagrange 插值**逼近分数阶延迟 $z^{-N_1}$，并将延迟**仅置于反馈回路**（Fig. 2b），得到改进型重复控制器：

$$\boxed{G_{rc}(z) = \frac{1}{1 - Q z^{-N_0} \sum\limits_{i=0}^{m} D_i z^{-i}} \cdot C(z)}$$

符号说明：

| 符号 | 含义 |
|---|---|
| $N_0 = \lfloor f_s / f_0 \rfloor$ | 整数阶延迟步数（$f_0$ 为基频） |
| $N_1 = N - N_0$ | 分数部分，$N_1 \in [0, 1)$ |
| $m$ | Lagrange 插值阶数（$m+1$ 个节点 $x_i = 0,1,\dots,m$） |
| $D_i = \prod\limits_{\substack{j=0 \\\\ j \neq i}}^{m} \dfrac{N_1 - j}{i - j}$ | Lagrange 插值系数，$\sum D_i z^{-i} \approx z^{-N_1}$ |
| $Q$ | 内模系数，典型值 $Q = 0.95$（见 Fig. 3 幅频特性分析） |
| $C(z)$ | 补偿器（相位超前 + 增益），保证闭环稳定性 |

**为什么将延迟仅置于反馈回路？** 传统结构中 $z^{-N}$ 同时出现在前向和反馈通路——前向延迟意味着当前周期的误差要到下一周期才产生补偿，限制了动态响应。改进结构（朱元 Fig. 2b）将延迟仅保留在反馈回路中，前向通路直接输入当前误差，在不损失谐波抑制能力的同时提升了动态性能。

## 与电机控制的关联

| 重复控制的特点 | 与电机控制的关联 |
|---|---|
| 谐波频率处无穷增益 | 直接针对转矩脉动的 $6k \pm 1$ 次谐波（5、7、11、13…） |
| 周期性扰动抑制 | 齿槽转矩、反电动势谐波、死区效应都是周期性的 |
| 无需精确被控对象模型 | 对参数变化鲁棒，与鲁棒控制有理论交集 |
| 计算量：需存储 $N$ 个历史误差值 | 高转速下 $N$ 小，低转速下 $N$ 大，需权衡内存 |

## 常见改进变体

### 1. 奇次谐波重复控制（Odd-Harmonic RC）

$$G_{rc,odd}(z) = \frac{-z^{-N/2}}{1 + z^{-N/2}}$$

只抑制奇次谐波（5、7、11、13…），**延迟减半**（$N/2$ 而非 $N$），动态响应更快。非常适合三相电机——因为对称结构中偶次谐波通常不存在。

### 2. $nk \pm m$ 次谐波重复控制

针对特定谐波族（如双端口机器的特定谐波模式）。

### 3. 分数阶重复控制

当采样频率不是基频的整数倍时（变频驱动中常见）。核心思路：$N = N_0 + N_1$，用 **Lagrange 插值**（朱元等，2024）或几何约束优化逼近分数延迟 $z^{-N_1}$。详见上文"离散实现"。

### 4. 自适应重复控制

频率自适应——对于变频电机驱动非常有用。
